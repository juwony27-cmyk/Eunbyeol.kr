<h1 align="center">Eunbyeol.kr</h1>

<p align="center">
  백은별 작가의 작품과 활동을 소개하고, 기록으로 남기는 팬 페이지 & 위키 프로젝트
</p>

<br>

**Eunbyeol.kr**은 백은별 작가와 관련된 콘텐츠를 소개하고 정리하기 위해 제작된 웹 프로젝트입니다.

이 프로젝트는 단순히 정보를 나열하는 페이지가 아니라,  
작가의 작품, 활동, 분위기, 인상을 시각적으로 전달하는 **팬 페이지**와  
관련 정보를 문서 형태로 정리하는 **Byeolpedia 위키**를 함께 운영하는 구조를 가지고 있습니다.

> 이 리포지토리는 [LICENSE](https://github.com/Anion15/Eunbyeol.kr/blob/main/LICENSE)로 보호됩니다.<br>
> 사용, 복사, 수정, 배포, 서브라이선스 관련 문의는 `juwony27@gmail.com`으로 부탁드립니다.

<br>

## Service Structure ㆍ Notice

| Service | Role | Description |
|---|---|---|
| [www.eunbyeol.kr](https://www.eunbyeol.kr) | Fan Page / Portfolio | 백은별 작가의 작품, 활동, 분위기, 소개 요소를 시각적으로 보여주는 메인 페이지 |
| [wiki.eunbyeol.kr](https://wiki.eunbyeol.kr) | Byeolpedia | 백은별 작가 관련 정보, 작품, 활동, 연혁 등을 정리하는 문서형 아카이브 |
<br>

본 리포지토리는 `wiki.eunbyeol.kr` > **Byeolpedia**의 서버 및 클라이언트 코드를 포함하지 않습니다.

이 저장소는 `www.eunbyeol.kr` > **Fan Page / Portfolio**의 서버 및 클라이언트 코드를 포함합니다.

즉, 본 리포지토리의 공개 범위는 **Eunbyeol.kr 메인 팬 페이지 및 포트폴리오 사이트**에 한정되며,  
문서형 아카이브인 **Byeolpedia 위키의 소스코드와 서버 구성은 별도로 관리**됩니다.

<br>

<br>

## Concept

> 소개는 감성적으로, 기록은 체계적으로.

Eunbyeol.kr은 하나의 도메인 아래에서  
**시각적 소개 페이지**와 **문서형 아카이브**를 분리하여 운영합니다.

- `www`는 분위기와 인상을 전달하는 소개용 페이지입니다.
- `wiki`는 정보를 정리하고 보존하는 기록용 공간입니다.
- 팬 페이지와 위키를 함께 제공하여, 감성적인 소개와 체계적인 기록을 동시에 목표로 합니다.
